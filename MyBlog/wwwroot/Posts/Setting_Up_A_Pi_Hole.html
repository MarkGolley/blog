What is DNS Resolution?

It‚Äôs how your devices translate human-friendly domain names (like bbc.co.uk) into IP addresses. Pi-hole steps in here to block unwanted domains at this point in the process.

Diagnosing Why It Was Inactive

After digging into the dashboard settings, I saw that it was using the Ethernet interface. I needed to switch it to use Wi-Fi.

I tried:

But it skipped the configuration menu, so I instead edited:

And changed:

Still no luck. I checked if DNS was running with:
